## 프로젝트 이름
- X스페이스

## 프로젝트 설명
- 200개 정도의 보드게임 만들기 (행사용)

  - 로그인

    - 아이디 및 비밀번호 찾기
  - 로그아웃
  - 회원 정보 확인

    - 레벨
    - 능력치
    - 랭킹
    - 경험치

      - 북마크 모음 리스트
  - 회원 정보 수정

    - 비밀번호
    - 닉네임

  - 관전 기능

  - 승리 화면
