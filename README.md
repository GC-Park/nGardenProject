## 프로젝트 이름
- X스페이스

## 프로젝트 설명
- 200개 정도의 보드게임 만들기 (행사용)

## 요구 사항 기능 목록
- 사용자 관련 사항
  - 로그인 없이 게스트 입장
    - 추후 회원가입 시 게스트 데이터 연동 (ex. Google, Facebook 등)
  - 회원가입
    - 본인인증
    - 아이디
    - 비밀번호
    - 닉네임
    - 성별
    - 나이
    - SNS 연동 회원가입
  - 회원탈퇴
    - SNS 연동 해제
  - 로그인
    - SNS 연동 로그인
    - 아이디 및 비밀번호 찾기
    - OTP 2차 인증
    - RECAPTCHA
  - 로그아웃
  - 마이페이지
    - 닉네임
    - 성별
    - 나이
    - 업적 기록
      - 업적에 따라 훈장 및 배지 지급
    - 전적(승/패 횟수)
    - 레벨
    - 능력치
    - 랭킹
    - 경험치
    - 관심있는 보드게임 리스트
    - 보유한 게임
    - 보유한 아이템
    - 보유한 포인트
  - 회원 정보 수정
    - 본인인증
    - 비밀번호
    - 닉네임
    - 성별
    - 나이
    - 관심 장르(게임)
    - 본인 소개
    - 프로필 공개 여부
  - 마이룸
    - 아바타
    - 훈장 및 배지
    - 공개/비공개
  - 펫
    - 부가 효과 부여 (ex. 획득 경험치 증가)
    - 관련 도전과제, 업적
  - 퀘스트
    - 퀘스트 완료 보상
    - 일일 퀘스트
    - 주간 퀘스트
    - 월간 퀘스트
  - 쪽지
    - 필터링
    - 쪽지함
      - 받은 쪽지함
      - 보낸 쪽지함
  
- 게임 관련
  - 게임 설명
    - 튜토리얼 기능
    - 도움말 화면
  - 싱글 플레이 (AI 대전)
    - 난이도 선택 가능
  - 멀티 플레이 (불특정 다수와 게임할 경우)
    - 빠른 시작 (랜덤 매칭)
    - 로비에서 직접 찾아 시작
  - 프리 플레이 (친구들과 게임할 경우)
    - 전적, 랭킹 미 반영
  - 게임 시작 및 진행
    - 플레이어 턴 표시
    - 턴 제한 시간 표시
    - 게임 이벤트 감지 및 동작 관리
    - 중도 퇴장
      - 중도 퇴장 시 불이익 부여 (ex. 전적 최하위 고정, 10분 간 멀티플레이 불가)
  - 게임 종료
    - 방으로 돌아가거나 퇴장
    - 게임 결과는 게임 마다 형식에 맞는 폼으로 출력
  
- 통계
  - 랭킹 기능
    - 전적
    - 레벨
  - 인기 게임
    - 플레이 횟수
    - 구매 유저 수
    - 일 단위
    - 주 단위
    - 월 단위
    - 연 단위
    - 전체
  
- 채팅
  - 음성 채팅
    - 온/오프 가능
  - 문자 채팅
    - 같이 플레이 중인 인원과 채팅
    - 친구 채팅
    - 길드 채팅
    - 귓속말
    - 확성기 같은 전체 채팅
    - 부적절한 채팅 필터링
  
- 공식 경기 관리
  - 리그 형식
    - 리그 순위 표
  - 토너먼트 형식
    - 토너먼트 대진표
  - 경기 스케줄
    - 일 단위
    - 주 단위
    - 월 단위
  - 승부 예측
    - 유저들은 전적과 랭킹을 기반으로 승/패를 예측
    - 맞히면 혜택 제공, 틀려도 불이익 없음
  
- 로비
  - 방 생성
    - 인원 설정
    - 게임 선택
    - 방 제목 설정
    - 방 비밀번호 유/무
    - 턴 제한 시간 설정
    - 방장
      - 강퇴 권한
      - 방 수정 권한
      - 시작 권한
    
  - 방 목록
    - 목록 필터링 및 검색 (장르 별, 등급 별, 인원 별)
    - 플레이 여부 확인
    - 입장 및 관전 가능 여부 확인
    - 입장 및 퇴장
      - 유저 초대하기
      - 모두가 퇴장하면 방은 자동으로 삭제
    - 관전 기능
      - 응원 기능

- 친목 
  - 친구
    - 선물하기
    - 온/오프라인 여부 리스트 확인
  - 길드
    - 길드 생성
      - 길드 소개 및 공지
    - 길드 가입
    - 길드 제명
    - 길드 초대
    - 길드 해체

- 설정
  - 사운드 설정
    - 배경음
    - 효과음
  - 그래픽 설정
    - 텍스쳐
  - 키보드 설정
  - 로그 설정
    - 리플레이는 로그 기반 재현 형식
  - 차단 설정
    - 유저 차단
    - 게임 차단

- 운영
  - 권장 사양 안내
  - 공지사항 안내
    - 상단 알림바로도 안내
    - 업데이트 내역 안내
    - 이벤트 안내 (ex. 이벤트로 각종 아이템 습득)
  - 오늘의 추천 게임 (무작위)
  - 건의 및 제보 기능
  - 신고 기능
  - 출석체크
    - 접속일에 따라 보상 지급
  - 블랙 리스트
    - 신고 당한 횟수
    - 중도 포기 횟수
